# Project: Global Video Production & OTT Streaming Platform

## 📋 Project Overview
Quality Assurance for a world-leading video production and streaming service. This project focused on the integrity of the subscription lifecycle, payment gateway integration, and user entitlement verification across multiple platforms.

* **Platform:** Web / Mobile App
* **Testing Type:** Test Case Execution, Functional Testing, Regression
* **Focus Area:** Subscription & Billing Workflows

---

## 🛠 Testing Activities
* **Test Case Execution:** Performed end-to-end verification of the subscription purchase flow, including trial activation and premium tier upgrades.
* **Billing Logic:** Validated payment processing success paths and UI/UX feedback during the checkout journey.
* **Access Control:** Confirmed that user accounts correctly unlocked premium content immediately after successful transaction verification.
* **Negative Scenarios:** Observed system behavior during payment cancellations or expired session attempts.

---

## 📊 Key Results
* **Test Cycle Compliance:** Successfully executed and documented comprehensive test suites within strict timeframes.
* **High-Stakes Verification:** Confirmed the stability of the revenue-generating core of the application (Billing & Subscriptions).
* **Cross-Platform Consistency:** Verified that subscription status synchronized accurately between mobile and desktop environments.

---

## 🎯 Key Achievements
* Applied a systematic approach to ensure **zero-failure** in the critical path of the user journey (checkout).
* Provided detailed evidence (logs/screenshots) for each step of the test case, ensuring full traceability for the development team.
* Demonstrated the ability to work within the high-security environment of a global media giant's financial systems.
