# Project: Global Video Production & OTT Streaming Platform

## 📋 Project Overview
Quality Assurance for a world-leading video production and streaming giant. The project focused on the integrity of the subscription lifecycle, payment gateway integration, and user entitlement verification.

* **Platform:** Web / Smart TV / Mobile
* **Testing Type:** Test Case Execution & Functional Testing
* **Focus Area:** Subscription & Billing Workflows

---

## 🛠 Testing Areas
* **Subscription Lifecycle:** End-to-end verification of trial activation, premium tier upgrades, and plan cancellations.
* **Payment Integration:** Testing the security and responsiveness of payment gateway UI/UX.
* **Access Control:** Ensuring instant content unlocking upon successful transaction verification.
* **Account Management:** Verifying cross-platform synchronization of subscription status.

---

## 🎯 Key Achievements
* Guaranteed 100% stability in the critical purchase path (checkout) by executing rigorous test suites.
* Verified billing logic for a global user base, ensuring correct currency and plan display.
* Demonstrated high-precision testing in a secure, high-stakes financial environment for a Tier-1 media provider.
